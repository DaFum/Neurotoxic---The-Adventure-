# Dokumentation

Die Dokumentation für **NEUROTOXIC: Grind The Void** liegt als einfache Markdown-Dateien im `docs/`-Ordner vor.

Da es sich um reines Markdown handelt, wird kein separater Build-Schritt (wie Sphinx oder MkDocs) benötigt. Du kannst die Dokumentation nativ auf GitHub, in deinem Editor (z.B. VS Code) oder direkt über das Dateisystem lesen.
